This is frontend based on React library

Objective: Build a collaborative task management application where users can create, assign and track tasks within different projects.

Feature:
  1. Interactive design for better user experience
  2. Role based access control
  3. Having different phases of task: todo, inprogress and done
  4. Team members can move their task to next step
  5. Activity logger to see the changes
  6. Kanban board to visualization of tasks steps
  7. Projects and task associated with can be seen on at glance window
  8. Project manager and admin can manage the team member profiles
  9. Insightful dashboard for keeping view on trends of operations
  10. Responsive and understandable user interface for ease of access.
